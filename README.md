# 📊 Shop Sales Dashboard (2023–2024) | Power BI Project

This repository contains a **Power BI interactive dashboard** that analyzes shop sales data from **2023 to 2024**, providing valuable business insights through rich visualizations.

---

## 🔍 Project Overview
The dashboard helps in understanding:
- Customer purchasing patterns  
- Product performance  
- Regional sales distribution  
- Monthly sales trends  

It enables **data-driven decision-making** through interactive filters and KPIs.

---

## ✨ Key Features
- Dynamic slicers for **Product, Region, and Item Cost**
- KPI cards for **Total Customers, Salespersons, and Sales**
- Interactive charts:
  - Line charts
  - Bar graphs
  - Pie & donut charts
- Clean and professional dashboard layout

---

## 🛠 Tools & Technologies
- **Power BI Desktop**
- **Microsoft Excel** (Data Source)
